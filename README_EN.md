## What is an Airport?

An "airport" is a proxy/VPN service provider offering subscription-based access to proxy nodes.

### Airport vs Self-Hosted VPS

| Factor | Airport | Self-Hosted VPS |
|--------|---------|-----------------|
| Price | $2-10/month | $5-20/month |
| Ease of Use | Ready to use | Requires setup |
| Stability | Provider dependent | VPS dependent |
| Nodes | Many locations | Self-managed |
| Unblocking | Usually better | Requires config |

## How to Choose

### Selection Criteria

1. **Stability** - Consistent service without drops
2. **Speed** - Sufficient bandwidth for streaming
3. **Nodes** - Number and locations available
4. **Unblocking** - Netflix/Disney+ capability
5. **Support** - Customer service quality
6. **Reputation** - User reviews and community feedback
