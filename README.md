## 什么是机场？

机场是提供VPN/代理服务的商家，用户通过订阅机场的服务来获得节点访问权限。

### 机场 vs 自建VPS

| 对比项 | 机场 | 自建VPS |
|--------|------|---------|
| 价格 | 月付10-50元 | 月付20-100元 |
| 易用性 | 开箱即用 | 需要配置 |
| 稳定性 | 依赖服务商 | 依赖VPS商家 |
| 节点数量 | 多地区可选 | 需要自己购买 |
| 解锁能力 | 通常更好 | 需要配置 |

## 如何选择机场

### 选择标准

1. **稳定性** - 服务是否稳定，频繁掉线不可取
2. **速度** - 带宽是否足够，看视频不卡顿
3. **节点** - 节点数量和地区是否符合需求
4. **解锁** - Netflix/Disney+等流媒体解锁能力
5. **售后** - 客服响应速度和技术支持
6. **口碑** - 用户评价和社区反馈

### 重要指标

- **延迟** - 200ms以内为优秀
- **带宽** - 100Mbps以上为佳
- **流量** - 月流量100GB以上
- **协议** - 支持Shadowsocks/V2Ray/Trojan等

## 常见问题

### Q: 机场安全吗？

A: 选择正规机场服务商是安全的。建议：
- 使用独立密码
- 开启两步验证
- 定期更换密码

### Q: 机场会被封吗？

A: 任何VPN服务都有被封风险。选择有多节点备用的机场，被封后可以快速切换。

### Q: 如何续费？

A: 一般通过官网后台自助续费，支持支付宝/微信等支付方式。

### Q: 可以多人使用吗？

A: 查看套餐的设备限制，一般个人套餐允许3-5个设备同时在线。

### Q: 退款政策？

A: 正规机场一般支持7天无理由退款，具体看各服务商政策。

## 避坑指南

### ❌ 避开的类型

1. **超低价套餐** - 低于市场价的可能有风险
2. **无限流量** - 物理带宽有限，不可能有真正无限
3. **无客服** - 出问题无法联系解决
4. **无名小站** - 无口碑、无社区评价

### ✅ 正确做法

1. 先看社区评价
2. 先购买月付试用
3. 确认退款政策
4. 测试节点速度和解锁

### 识别劣质机场

- 网站粗糙、无备案信息
- 价格极低、承诺无限
- 无客服或客服不专业
- 用户反馈差、经常掉线
